# Automatic Tool Usage

Use the best available tool without being asked. If a tool/MCP/agent/skill/command exists that would produce a better result — USE IT.

## MCPs — Auto-Use

| MCP | When |
|-----|------|
| GitHub | PRs, issues, reviews, CI status |
| Memory | "remember this", persistent context |
| Sequential Thinking | Complex multi-step reasoning |
| context7 | Up-to-date library/framework docs |

## Plugins — Auto-Use

| Plugin | When |
|--------|------|
| pyright-lsp / typescript-lsp | Writing or editing code (type errors, go-to-definition) |
| mgrep | Codebase search, cross-file pattern matching |
